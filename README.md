# Customer Behavior Analysis Dashboard

## 📌 Project Overview

Customer behavior plays a critical role in business growth and decision-making. Understanding how customers purchase products, subscribe to services, choose payment methods, and respond to seasonal trends enables businesses to improve customer satisfaction and increase revenue.

This project analyzes customer shopping behavior using Python and Power BI. The dataset was first cleaned and explored in Google Colab using Python libraries, then transformed into an interactive Power BI dashboard that provides actionable business insights.

The dashboard helps businesses answer questions such as:

* Which product categories generate the highest revenue?
* What products are purchased most frequently?
* How many customers are subscribed?
* Which payment methods are preferred?
* How does revenue vary across seasons?
* What opportunities exist to improve customer retention and sales?

---

## 🎯 Business Problem

Businesses collect large amounts of customer transaction data but often struggle to convert it into meaningful insights.

Without proper analysis, businesses may:

* Miss customer retention opportunities
* Invest in low-performing products
* Fail to identify seasonal demand patterns
* Overlook customer preferences
* Make decisions based on assumptions rather than data

This project solves these challenges by providing a data-driven customer behavior analysis solution.

# 🔍 Data Cleaning Process (Python)

Before visualization, the dataset was cleaned to improve quality and reliability.

### Data Cleaning Steps

✔ Imported dataset using Pandas

✔ Checked dataset dimensions and structure

✔ Identified missing values

✔ Removed duplicate records

✔ Validated categorical values

✔ Converted columns to appropriate data types

✔ Checked outliers and inconsistencies

✔ Performed data quality validation

### Sample Cleaning Code

```python
import pandas as pd

df = pd.read_csv("customer_shopping_data.csv")

# Check missing values
df.isnull().sum()

# Remove duplicates
df = df.drop_duplicates()

# Convert date column
df["Purchase Date"] = pd.to_datetime(df["Purchase Date"])

# Verify data types
df.info()
```
